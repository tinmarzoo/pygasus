# PYGASUS

Pygasus is a grade scraping tool for pegasus website.


### Dependencies

* [RoboBrowser] - python web scraping tool
* [python 3] - python 3+

### Installation

```sh
$ git clone https://github.com/tinmarzoo/pygasus.git
```
### Usage

```sh
$ python pygasus
```

see
```sh
$ python pygasus -help 
```
for help
